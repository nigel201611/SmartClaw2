欢迎使用 SmartClaw

版本 2.0.0 Standalone

🚀 本地 Matrix 服务器

内置 Conduit Matrix 服务器，无需外部服务即可运行。

💬 即时通讯

支持 1:1 聊天、群聊、文件传输、消息历史等完整功能。

🔒 端到端加密

所有消息默认启用 E2E 加密，保护您的隐私。

☁️ 离线优先

消息本地存储，离线也能查看历史记录。

⚙️ 系统要求

  • Docker Desktop - 必须安装并运行
  • macOS 10.15+ / Windows 10+ / Linux
  • 内存 至少 4GB（推荐 8GB）
  • 磁盘空间 至少 1GB 可用空间

首次启动时，SmartClaw 会自动创建并启动 Matrix 容器。

如遇到问题，请查看应用内日志或访问文档。
