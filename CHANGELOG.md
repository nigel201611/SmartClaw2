## [2.0.0] - 2026-03-13

### 🎉 新增 - Major Release

#### 核心功能
- **内置 Matrix 服务器**
  - 集成 Conduit Matrix 服务器（Rust，轻量级）
  - Docker 容器自动管理
  - 无需外部 Matrix 服务器即可运行
  
- **全新聊天界面**
  - 现代化响应式设计
  - 消息气泡（发送/接收样式区分）
  - Markdown 格式支持（粗体/斜体/代码/链接）
  - 实时输入状态指示器
  - 自动滚动到最新消息
  - 日期分组显示

- **用户认证系统**
  - 登录/登出流程
  - 会话恢复（自动登录）
  - 凭证安全存储（系统密钥链）
  - 记住我选项
  - 密码可见性切换

- **消息持久化**
  - SQLite 本地存储
  - 离线消息查看
  - 增量同步（5 分钟间隔）
  - 冲突解决策略
  - 已读状态跟踪

- **Docker 集成**
  - 4 步 Docker 环境检测
  - 容器自动启动/停止
  - 健康检查（60 秒超时）
  - 资源限制配置（CPU/内存）
  - 系统托盘控制

- **设置管理**
  - 8 个可配置项
  - 设置持久化
  - 导入/导出功能
  - 恢复默认设置

#### 技术特性
- **跨平台支持**
  - macOS (DMG + ZIP, x64 + arm64)
  - Windows (NSIS + Portable)
  - Linux (AppImage + deb)

- **安全性**
  - 系统密钥链集成（keytar）
  - hardened runtime (macOS)
  - 代码签名 + 公证
  - 端到端加密支持

- **性能优化**
  - 内存占用 <300MB
  - 冷启动 <12 秒
  - 消息延迟 <100ms
  - WAL 模式数据库

## 版本说明

### 语义化版本
- **MAJOR.MINOR.PATCH** (主版本号。次版本号。修订号)
- **MAJOR**: 不兼容的 API 变更
- **MINOR**: 向后兼容的功能新增
- **PATCH**: 向后兼容的问题修正

### 发布周期
- **Major**: 每 6-12 个月
- **Minor**: 每 1-2 个月
- **Patch**: 按需发布

### 支持政策
- **v2.0**: 支持至 2027-03-13
- **v1.0**: 已停止支持（2026-03-13）
