## 🚀 Quick Start

### 1. Review Generated Configurations

All configurations have been generated based on your environment detection results.

### 2. Set Up Environment Variables

```bash
# Copy the template
cp .env.template .env

# Edit with your actual values
nano .env
```

### 3. Deploy Services

#### Option A: Docker Deployment

```bash
docker-compose up -d
```

#### Option B: Manual Deployment

1. **Start Higress Gateway**
   ```bash
   # See Higress documentation for installation
   ```

2. **Start Matrix Server**
   ```bash
   # Using the generated homeserver.yaml
   synapse_port 8008 --config-path matrix/homeserver.yaml
   ```

3. **Start MinIO**
   ```bash
   docker run -d -p 9000:9000 minio/minio server /data
   ```

### 4. Start SmartClaw Agent

```bash
cd apps/agent
npm install
npm start
```

### 5. Start Desktop App

```bash
cd apps/desktop
npm install
npm start
```

## 🔧 Configuration Files

### openclaw.json

Main agent configuration including:
- Model endpoint settings
- Matrix server connection
- Storage configuration
- Logging settings

### higress/config.yaml

Gateway routing configuration for:
- LLM provider routes
- Matrix server proxy
- MinIO access

### matrix/homeserver.yaml

Matrix Synapse homeserver configuration:
- Server name and URLs
- Database settings
- Registration settings
- Rate limiting

## 🔐 Security Notes

- **Never commit** the `credentials/` directory to version control
- Store the `.key` file securely (consider using a secrets manager)
- Rotate credentials regularly
- Use HTTPS in production

## 📝 Regenerating Configurations

To regenerate configurations:

```bash
cd packages/config-gen
npm start init
```

## ❓ Troubleshooting

### Port Conflicts

If you encounter port conflicts, run:

```bash
npm start detect
```

This will show which ports are available.

### Invalid Configurations

Validate configurations:

```bash
npm start validate
```
